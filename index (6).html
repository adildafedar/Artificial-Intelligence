Video Frame Extraction and Segmentation for Real-Time Object Analysis

2

Problem

Videos contain lots of information but are hard to analyze manually. Detecting and identifying objects in videos automatically is challenging, especially in real-time. Also, linking detected objects to online shopping options is not widely available.

Solution

A system that extracts frames from videos, uses deep learning (YOLOv8n) to detect and segment objects, and automatically generates product search links for detected items.

3

Aim

To create an automated system that processes video frames, detects objects accurately, and helps users find related products online quickly.

Objectives

  • To extract frames from real-time or pre-recorded video at a defined frame rate.
  • To apply object detection and segmentation models to identify objects in each frame.
  • To filter and list unique objects detected across the video.
  • To generate automated purchase/search links for detected products using online platforms like Amazon.

Methodology

  1. Frame Extraction
    Input: Pre-recorded video (input_video.mp4).
    Frames are extracted at 2 FPS using OpenCV.
    Each frame is saved sequentially as an image file for further processing.
  2. Object Detection
    Model: YOLOv8n.pt from the Ultralytics YOLO library.
    Each saved frame is passed to the YOLO model for detection.
    Objects detected are collected across frames and filtered for uniqueness.
    Detections are logged and stored in detected_objects.txt.
  3. Product Link Generation
    Each unique object is mapped to a product search query.
